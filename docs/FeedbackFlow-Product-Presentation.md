## 4. How It Works

### The Complete Workflow

#### Step 1: Setup Your Organization (One-Time)

**For Administrators:**
1. Create your organization account
2. Import users via CSV or add manually
3. Set up organizational hierarchy (who reports to whom)
4. Create departments and teams
5. Assign user roles (Admin, Manager, Employee)

**Time Required:** 1-2 hours for initial setup

---

#### Step 2: Create Feedback Templates (Reusable)

**For HR/Administrators:**
1. Define competencies to rate (e.g., Communication, Teamwork)
2. Add open-ended questions
3. Set up goal-setting fields
4. Save as template for future use

**Example Templates:**

- Quarterly Check-in
- 360Â° Feedback
- Manager Evaluation
- Self-Assessment

#### Step 3: Launch a Performance Cycle

**For Managers/HR:**
1. Click "Create New Cycle"
2. Choose dates (start, end, deadline)
3. Select template
4. Define participants:
   - Entire organization
   - Specific departments
   - Individual teams
   - Custom groups
5. Activate cycle â†’ Automatic notifications sent

**Cycle Types:**
- **Manager to Employee**: Traditional performance review
- **360Â° Feedback**: Input from peers, direct reports, manager

- **Peer Review**: Team members evaluate each other

---

#### Step 4: Give Feedback

**For Employees/Managers:**
1. Receive notification about feedback request
2. Log in to FeedbackFlow
3. Open feedback form
4. Rate competencies (1-5 scale with descriptions)
5. Write detailed comments
6. Set goals for next period (if applicable)

**Key Features:**
- Auto-save prevents data loss
- Draft system allows working over multiple sessions
- Clear guidance on rating criteria
- Word count suggestions for comments

#### Step 5: Review & Acknowledge

**For Feedback Recipients:**
1. Receive notification when feedback is submitted
2. Log in and review feedback
3. Read ratings and comments
4. Acknowledge receipt

#### Step 6: Monitor Progress

**For Managers/HR:**
- View real-time completion dashboard
- See who has/hasn't submitted
- Send reminders to specific individuals
- Track approaching deadlines
- Export data for offline analysis

---

#### Step 7: Analyze & Act

**For Leadership/HR:**
